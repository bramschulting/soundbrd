# Soundbrd

Soundbrd is a CLI soundboard.

## Instalation

`npm install soundbrd -g`

## Usage

```
Usage: soundbrd <command>

Add sounds:
    soundbrd add <name> <path>

    Example:
    soundbrd add foo foo.mp3

    Alias(es):
    a
Play sounds:
    soundbrd add <id|name>

    Example:
    soundbrd play foo

    Alias(es):
    p
List sounds:
    soundbrd list

    Alias(es):
    l, ls
Remove sounds:
    soundbrd delete <id|name>

    Example:
    soundbrd delete foo

    Alias(es):
    d
```
